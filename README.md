# 在线学习Flutter应用

这是一个使用Flutter开发的在线学习应用，支持多平台部署。

## 项目特点

- 使用GetX进行状态管理和路由
- 支持多语言国际化
- 自适应主题切换
- 完整的用户认证系统
- 课程浏览和学习功能

## 技术栈

- Flutter SDK: ^3.6.2
- GetX: ^4.7.2
- Dio: ^5.8.0+1
- 其他依赖请查看pubspec.yaml文件

## 开始使用

1. 克隆仓库
2. 运行 `flutter pub get` 安装依赖
3. 运行 `flutter run` 启动应用

## 项目结构

- lib/common: 公共组件、工具和服务
- lib/models: 数据模型
- lib/pages: 应用页面
